# Cookies — Exemplo de tema (claro/escuro)

Este diretório contém um exemplo didático que mostra como usar cookies
para salvar a preferência de tema (modo claro / modo escuro) e aplicá-la
na interface do usuário usando HTML, CSS e JavaScript.

Arquivos:
- [cookies/index.html](cookies/index.html)
- [cookies/style.css](cookies/style.css)
- [cookies/script.js](cookies/script.js)

Como usar:
1. Abra [cookies/index.html](cookies/index.html) em um navegador.
2. Selecione um tema no menu (Claro, Escuro, Roxo, Rosa, Azul, Azul bebê, Laranja, Verde, Amarelo, Vermelho, Marrom).
3. As cores usam tons médios / pastel — não são muito claras nem muito escuras — para boa legibilidade.
4. A preferência será salva em um cookie válido por 1 ano (exceto "Claro", que é o padrão sem cookie).
5. Use o botão "Resetar preferência" para remover o cookie e voltar ao tema padrão.
